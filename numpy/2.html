Understanding np.array() in NumPy

By Manushi Katipearachchi · Aug 2, 2025

List

A list is a collection of items in square brackets.

[1, 2, 3]

Tuple

A tuple is a collection of items in round brackets.

(1, 2, 3)

🔸Both hold multiple values in order.
🔸Both can be used to build arrays.

Python List vs Tuple Comparison Table


Feature List Tuple Info
Mutability Lists can be changed (mutable) Tuples can’t be changed (immutable)
Syntax Square brackets [ ] Parentheses ( )
Performance Slower Faster
Methods More built-in methods Fewer methods
Use Cases For dynamic collections For fixed collections
Memory Usage More memory Less memory
Iteration Slightly slower Faster
Hashability Not hashable Hashable (if elements are hashable)

Why Use a List or Tuple

🔹 They group values together
🔹 They show shape and structure
🔹 NumPy reads the shape from the nesting
🔹 Lists are mutable, so you can change their contents
🔹 Tuples are immutable, which can help with data integrity
🔹 Tuples can be used as dictionary keys if their elements are hashable
🔹 Lists have many built-in methods for easy manipulation
🔹 Tuples use less memory and can be faster to iterate over

Why Use np.array() to Create Arrays?

np.array() is used to convert Python lists or tuples into NumPy arrays because these arrays:

🔹Store data efficiently in contiguous memory blocks, which speeds up computations.
🔹Allow fast, vectorized mathematical operations without writing loops.
🔹Support multi-dimensional data like matrices and tensors naturally.
🔹Use less memory compared to Python lists and provide better performance.
🔹Enable powerful features like broadcasting and many built-in scientific functions.

This makes np.array() essential for scientific computing, data analysis, and machine learning tasks.

How np.array() Works

np.array() takes one argument only. That argument must be a list, tuple, or a nested version of either.

✅ np.array([[1, 2], [3, 4]])  # Correct

🔸This is a list of lists ( [ [ ] , [ ] ] )
🔸NumPy turns it into a 2D array

❌ np.array([1, 2], [3, 4])  # Error

🔸This passes two arguments ( [ ] , [ ] )
🔸NumPy expects just one

Examples of Array Types


1D array

np.array([1, 2, 3])

2D array

np.array([[1, 2], [3, 4]])

3D array

np.array([[[1], [2]], [[3], [4]]])

So, always pass your data in one list or tuple. That’s how np.array() understands it 🙃
